# FirstWalletInstitute

FirstWalletInstitute is an open educational framework for teaching first-time digital earners how wallets, payment apps, Web3 domains, custody, safety, and offramps fit together.

This project began with a simple real-world question:

> How should a young person receive payment for helping with work?

The first answer may be Venmo or Cash App.

But that answer opens a larger doorway:

- What is a wallet?
- Who controls it?
- What happens if access is lost?
- Can a domain become a payment identity?
- How does value move from app to bank to blockchain and back?
- How can families teach this safely before large amounts of money are involved?

This repo is a starting point for that conversation.
